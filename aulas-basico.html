HTML Básico

As tags fundamentais que vais usar em absolutamente todas as páginas web.

0. A Estrutura Base do HTML5

Todo o ficheiro HTML começa com esta estrutura obrigatória. Sem ela, o browser pode interpretar mal o conteúdo.

<!DOCTYPE html>
<html lang="pt">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>O título que aparece na aba do browser</title>
  </head>
  <body>
    <!-- O teu conteúdo visível vai aqui dentro -->
  </body>
</html>

O que faz cada parte?

  • <!DOCTYPE html> → Diz ao browser que estamos a usar HTML5.
  • <html lang="pt"> → Raiz de tudo. O atributo lang indica o idioma.
  • <head> → "Cérebro" invisível: título da aba, ligações a CSS, configurações.
  • <meta charset="UTF-8"> → Garante que os caracteres especiais (ã, ç, é...) aparecem corretamente.
  • <meta name="viewport"...> → Faz a página funcionar bem em telemóveis.
  • <body> → Tudo o que o utilizador vê vai aqui.

1. Tags de Título (h1 a h6)

Existem 6 níveis de títulos em HTML. O <h1> é o mais importante e o <h6> o menos. Devem ser usados de forma hierárquica — como um índice de livro.

<h1>Título Principal (apenas um por página)</h1>
<h2>Título de Secção</h2>
<h3>Título de Sub-secção</h3>
<h4>Título de nível 4</h4>
<h5>Título de nível 5</h5>
<h6>Título de nível 6 (o mais pequeno)</h6>

h1 — Título Principal

h2 — Título de Secção

h3 — Sub-secção

h4 — Nível 4

h5 — Nível 5
h6 — Nível 6
💡 Cada página deve ter apenas um <h1>. É o título principal — como o título de um livro.

2. Parágrafos e Texto

Para qualquer bloco de texto corrido, usa a tag <p> (de Paragraph). O browser adiciona automaticamente espaço antes e depois de cada parágrafo.

<p>Este é o meu primeiro parágrafo. Pode ter o texto que quiseres.</p>

<p>Este é o segundo parágrafo. O browser separa-os automaticamente.</p>
⚠️ Não uses <br> em excesso para criar espaços entre parágrafos. Usa vários <p> — é a forma correta!

3. Ênfase: Negrito, Itálico e Sublinhado

Existem tags para dar destaque a partes do texto. É importante conhecer a diferença entre as versões semânticas e as visuais:

TagResultadoSignificado
<strong>Texto em negritoConteúdo importante (semântico)
<b>Texto em negritoApenas negrito visual (sem significado)
<em>Texto em itálicoÊnfase (semântico)
<i>Texto em itálicoApenas itálico visual (sem significado)
<u>Texto sublinhadoAnotação (usar com cautela)
<p>Esta palavra é <strong>muito importante</strong>.</p>
<p>Podes combinar: <strong><em>negrito e itálico</em></strong>.</p>
💡 Prefere sempre <strong> a <b>, e <em> a <i>.

4. Listas

Existem dois tipos de listas em HTML, e ambas usam a tag <li> (de List Item) para os itens:

Lista Não-Ordenada (ul) — com pontos

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Lista Ordenada (ol) — numerada

<ol>
  <li>Aprende HTML</li>
  <li>Aprende CSS</li>
  <li>Aprende JavaScript</li>
</ol>

Lista Dentro de Lista (aninhada)

<ul>
  <li>Linguagens da Web
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>JavaScript</li>
</ul>

5. Links

Os links são o que torna a web uma "teia". Usamos a tag <a> (de Anchor) com o atributo href.

<a href="https://google.com">Ir para o Google</a>
<a href="https://google.com" target="_blank">Abrir em nova aba</a>
<a href="sobre.html">Sobre nós</a>
<a href="mailto:email@exemplo.com">Envia-nos um e-mail</a>
<a href="#contacto">Ir para Contactos</a>
💡 target="_blank" abre o link numa nova aba. Para segurança, junta sempre rel="noopener noreferrer" quando ligares a sites externos.

6. Imagens

A tag <img> é auto-fechada — não precisa de </img>. Os atributos src e alt são obrigatórios.

<img src="imagens/foto.jpg" alt="Descrição da foto" width="300">
<img src="https://exemplo.com/imagem.png" alt="Descrição" width="400">
  • src → Caminho para a imagem (local ou URL).
  • alt → Texto que aparece se a imagem não carregar. Fundamental para acessibilidade.
  • width / height → Dimensões da imagem em píxeis.
⚠️ Nunca deixes o alt vazio se a imagem tiver conteúdo.

7. Quebras de Linha e Divisores

Duas tags simples mas úteis para separar conteúdo:

<p>Linha um.<br>Linha dois — sem espaço entre elas.</p>
<hr>
💡 Usa <hr> para separar secções de conteúdo temáticas.

8. Comentários

Os comentários são notas que escreves no código para ti mesmo (ou para outros programadores). O browser ignora completamente os comentários.

<!-- Isto é um comentário e não aparece na página -->

<!-- 
    Comentários podem ter
    múltiplas linhas
-->

<p>Isto aparece na página.</p>
<!-- <p>Este parágrafo está "desativado" pelo comentário.</p> -->
