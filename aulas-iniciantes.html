Teoria Fundamental

Antes de escrever código, é preciso entender o que estamos realmente a construir.

🌐 O que é a Internet?

A Internet é uma enorme rede global de computadores interligados entre si. Pensa nela como uma teia gigante que conecta milhões de máquinas em todo o mundo, permitindo a troca de informação a alta velocidade.

💡 Dica: A Internet e a Web não são a mesma coisa! A Internet é a infraestrutura (os "cabos e antenas"). A Web (World Wide Web) é um dos serviços que corre sobre a Internet — as páginas que visitas no browser.

A Internet surgiu nos anos 60/70, inicialmente como um projeto militar americano chamado ARPANET. Com o tempo, tornou-se pública e hoje é usada por mais de 5 mil milhões de pessoas.

🖥️ O que é o HTML?

A sigla HTML significa HyperText Markup Language — em português, Linguagem de Marcação de Hipertexto.

  • HyperText → Texto com links que nos transportam para outros documentos.
  • Markup → Marcação: usamos "etiquetas" para dizer ao browser o que cada coisa é.
  • Language → É uma linguagem com regras e sintaxe própria.
⚠️ Atenção: O HTML não é uma linguagem de programação. Não tem lógica, condições (if/else) ou variáveis. É uma linguagem de estrutura e conteúdo.

O HTML define o esqueleto de uma página web: onde está o título, onde está o texto, onde vai a imagem, onde começa o menu.

📖 A História do HTML

O HTML foi criado por Tim Berners-Lee, um cientista britânico que trabalhava no CERN, na Suíça.

  • 1989 — Tim Berners-Lee propõe um sistema de partilha de documentos científicos entre investigadores.
  • 1991 — É publicado o primeiro documento HTML. A World Wide Web nasce!
  • 1995 — HTML 2.0: primeira versão com formulários e tabelas.
  • 1999 — HTML 4.01: versão muito usada durante anos, trouxe CSS e JavaScript como parceiros.
  • 2014 — HTML5: a versão atual. Suporte nativo para vídeo, áudio, semântica e muito mais.

🏷️ O que são Tags?

As tags são as "etiquetas" do HTML. São instruções que dizem ao browser como deve interpretar e apresentar o conteúdo.

Uma tag é escrita entre os símbolos < e >. Quase todas têm uma tag de abertura e uma de fecho:

Tag de Abertura e Fecho

<p>Isto é um parágrafo.</p>
<h1>Isto é um título.</h1>
<strong>Isto está em negrito.</strong>

Tags Auto-Fechadas (sem conteúdo)

<br>     <!-- Quebra de linha -->
<hr>     <!-- Linha horizontal -->
<img>    <!-- Imagem -->
<input>  <!-- Campo de formulário -->
✅ Regra de ouro: Tudo o que abres, tens de fechar! <p> → </p>. A barra / indica que é a tag de fecho.

🔧 Atributos HTML

Os atributos são informações extra que colocamos dentro das tags para configurar o seu comportamento. Seguem sempre o formato nome="valor".

<!-- O atributo "href" define o destino do link -->
<a href="https://google.com">Ir para o Google</a>

<!-- O atributo "src" indica o caminho da imagem -->
<img src="foto.jpg" alt="Descrição da foto" width="300">

<!-- O atributo "class" serve para aplicar estilos CSS -->
<p class="destaque">Este parágrafo tem uma classe.</p>

Atributos mais comuns

  • href → Define o destino de um link.
  • src → Define o caminho de uma imagem ou ficheiro.
  • alt → Texto alternativo para imagens (acessibilidade).
  • class → Associa um estilo CSS ao elemento.
  • id → Identificador único de um elemento.
  • style → CSS diretamente dentro da tag (não recomendado).
  • target → Define como abrir um link (_blank = nova aba).

⚙️ Como o Browser Interpreta o HTML?

Quando abres uma página web, o browser (Chrome, Firefox, Edge...) faz o seguinte:

  1. Descarrega o ficheiro HTML do servidor.
  2. Lê o código de cima para baixo, linha a linha.
  3. Interpreta cada tag e constrói uma árvore chamada DOM (Document Object Model).
  4. Aplica os estilos CSS que encontrar.
  5. Renderiza (desenha) a página no ecrã.
💡 Podes ver os codigos do HTML em qualquer página! Clica com o botão direito no browser → "Inspecionar".

🎨 A Trindade da Web

Para criar uma página web completa, precisas de três tecnologias que trabalham juntas:

🦴

HTML

O esqueleto. Define a estrutura e o conteúdo.

🎨

CSS

A aparência. Define cores, tamanhos e layout.

⚡

JavaScript

O comportamento. Animações, cliques e interatividade.

🚀 Neste projeto vais aprender as três! Começas pelo HTML, depois CSS e por fim JavaScript.
