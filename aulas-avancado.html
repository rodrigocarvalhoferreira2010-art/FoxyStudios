Flexbox, Grid, responsividade, animações e os teus primeiros passos em JavaScript.

1. Flexbox — Layout em Uma Dimensão

O Flexbox é um sistema de layout CSS que permite organizar elementos numa linha ou coluna de forma flexível e responsiva. Aplica-se ao elemento pai.

.container {
    display: flex;              /* Ativa o Flexbox */
    flex-direction: row;        /* row (padrão) | column */
    justify-content: center;    /* Eixo principal: flex-start | center | flex-end | space-between | space-around */
    align-items: center;        /* Eixo cruzado: stretch | flex-start | center | flex-end */
    gap: 16px;                  /* Espaço entre os filhos */
    flex-wrap: wrap;            /* wrap = quebra linha se necessário */
}

.filho {
    flex: 1;                    /* Cresce igualmente para preencher o espaço */
}

2. CSS Grid — Layout em Duas Dimensões

O Grid é ideal para layouts complexos com linhas e colunas ao mesmo tempo — ideal para a estrutura geral de uma página.

.grelha {
    display: grid;
    grid-template-columns: repeat(3, 1fr);  /* 3 colunas iguais */
    grid-template-columns: 1fr 2fr 1fr;     /* 3 colunas com proporções diferentes */
    gap: 20px;                               /* Espaço entre células */
}

.elemento-largo {
    grid-column: span 2;     /* Ocupa 2 colunas */
}

.elemento-completo {
    grid-column: 1 / -1;     /* Ocupa a linha toda */
}

3. Media Queries — Responsividade

As Media Queries permitem aplicar CSS diferente consoante o tamanho do ecrã. É assim que uma página se adapta a telemóveis, tablets e computadores.

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

@media (max-width: 768px) {
    .container {
        grid-template-columns: repeat(2, 1fr);
    }
}

@media (max-width: 480px) {
    .container {
        grid-template-columns: 1fr;
    }
    .sidebar {
        display: none;
    }
}
💡 A abordagem Mobile First (desenhar primeiro para telemóvel, depois expandir para desktop) é a mais recomendada atualmente. Usa min-width em vez de max-width.

4. Variáveis CSS (Custom Properties)

As variáveis CSS permitem guardar valores (cores, tamanhos, fontes) numa só sítio e reutilizá-los em todo o projeto. Mudar a cor principal passa a ser uma alteração num único lugar.

:root {
    --cor-primaria: #ff4d4d;
    --cor-texto: #333;
    --cor-fundo: #f0f2f5;
    --raio-borda: 8px;
    --sombra: 0 4px 12px rgba(0,0,0,0.1);
}

.botao {
    background-color: var(--cor-primaria);
    border-radius: var(--raio-borda);
    box-shadow: var(--sombra);
    color: white;
}

h1 { color: var(--cor-primaria); }
body { color: var(--cor-texto); background: var(--cor-fundo); }

5. Transições e Animações CSS

Transições (transition)

Definem a suavidade da mudança entre dois estados (ex: cor normal → cor ao passar o rato).

.botao {
    background-color: #ff4d4d;
    transition: background-color 0.3s ease, transform 0.2s ease;
}

.botao:hover {
    background-color: #cc0000;
    transform: translateY(-3px);
}

Animações (animation + @keyframes)

Permitem criar animações complexas com vários estados intermédios.

@keyframes aparecer {
    from { opacity: 0; transform: translateY(20px); }
    to   { opacity: 1; transform: translateY(0); }
}

.card {
    animation: aparecer 0.5s ease forwards;
}

@keyframes rodar {
    from { transform: rotate(0deg); }
    to   { transform: rotate(360deg); }
}

.loading {
    animation: rodar 1s linear infinite;
}

6. Introdução ao JavaScript

O JavaScript é a linguagem de programação da web. Ao contrário do HTML e CSS, tem lógica real — variáveis, condições, loops e funções.

<script>

// VARIÁVEIS
let nome = "Rodrigo";
const idade = 15;

// TIPOS DE DADOS
let texto = "Olá!";
let numero = 42;
let booleano = true;
let lista = [1, 2, 3];
let objeto = { nome: "RC" };

// CONDIÇÕES
if (idade >= 18) {
    console.log("É maior de idade");
} else {
    console.log("É menor de idade");
}

// LOOP
for (let i = 0; i < 5; i++) {
    console.log("Número: " + i);
}

// FUNÇÃO
function saudar(nome) {
    return "Olá, " + nome + "!";
}
console.log(saudar("Rodrigo"));

</script>

7. Manipulação do DOM

O DOM (Document Object Model) é a representação em memória de toda a página HTML. Com JavaScript, podemos selecionar, modificar e criar elementos em tempo real.

// SELECIONAR ELEMENTOS
const titulo = document.getElementById("titulo");
const cards = document.querySelectorAll(".card");
const primeiro = document.querySelector(".card");

// MODIFICAR CONTEÚDO
titulo.textContent = "Novo Título!";
titulo.innerHTML = "<em>Título em itálico</em>";

// MODIFICAR ESTILOS
titulo.style.color = "red";
titulo.style.fontSize = "2em";

// ADICIONAR/REMOVER CLASSES
titulo.classList.add("destaque");
titulo.classList.remove("antigo");
titulo.classList.toggle("visivel");

// CRIAR E ADICIONAR ELEMENTOS
const novoParagrafo = document.createElement("p");
novoParagrafo.textContent = "Criado com JavaScript!";
document.body.appendChild(novoParagrafo);

8. Eventos JavaScript

Os eventos são ações do utilizador (clique, tecla, mover o rato...) às quais o JavaScript pode reagir.

const botao = document.getElementById("meu-botao");

botao.addEventListener("click", function() {
    alert("Clicaste no botão!");
});

botao.addEventListener("click", () => {
    document.getElementById("resultado").textContent = "Clicado!";
});

elemento.addEventListener("click", () => {});
elemento.addEventListener("mouseover", () => {});
elemento.addEventListener("mouseout", () => {});
input.addEventListener("input", () => {});
formulario.addEventListener("submit", (e) => {
    e.preventDefault();
});
