HTML Médio & Introdução ao CSS

Semântica, tabelas, formulários e os teus primeiros passos no CSS.

1. HTML Semântico — Tags com Significado

O HTML5 trouxe tags semânticas — tags cujo nome descreve o seu propósito. Ajudam o browser, os motores de busca e os leitores de ecrã a entender a estrutura da página.

<header>
  <nav>
    <a href="index.html">Início</a>
    <a href="sobre.html">Sobre</a>
  </nav>
</header>

<main>
  <section>
    <h2>Secção principal</h2>
    <article>
      <h3>Artigo individual</h3>
      <p>Conteúdo do artigo...</p>
    </article>
  </section>

  <aside>
    <p>Conteúdo lateral (publicidade, links relacionados)</p>
  </aside>
</main>

<footer>
  <p>© 2025 Rodrigo. Todos os direitos reservados.</p>
</footer>
<header>

Cabeçalho da página ou de uma secção. Geralmente contém o logótipo e navegação.

<nav>

Bloco de navegação. Links principais do site.

<main>

Conteúdo principal da página. Deve ser único por página.

<section>

Secção temática de conteúdo. Agrupa conteúdo relacionado.

<article>

Conteúdo independente (post de blog, notícia, comentário).

<aside>

Conteúdo lateral, complementar ao principal.

<footer>

Rodapé da página. Contactos, links legais, copyright.

<div>

Divisão genérica sem significado semântico. Usar quando nenhuma outra tag serve.

2. Tabelas

As tabelas servem para organizar dados em grelha (linhas e colunas). Não devem ser usadas para layout de página!

<table>
  <thead>
    <tr>
      <th>Nome</th>
      <th>Idade</th>
      <th>País</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Rodrigo</td>
      <td>15</td>
      <td>Portugal</td>
    </tr>
  </tbody>
</table>
  • <table> → Contentor da tabela toda.
  • <thead> → Cabeçalho da tabela (linha com os títulos das colunas).
  • <tbody> → Corpo da tabela (os dados).
  • <tr> → Table Row — uma linha da tabela.
  • <th> → Table Header — célula de cabeçalho (negrito e centrado por defeito).
  • <td> → Table Data — célula de dados normal.

3. Formulários

Os formulários permitem ao utilizador introduzir dados — como campos de login, pesquisa, ou registo.

<form action="processar.php" method="POST">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome" placeholder="O teu nome">

  <label for="email">E-mail:</label>
  <input type="email" id="email" name="email">

  <label for="pass">Palavra-passe:</label>
  <input type="password" id="pass" name="pass">

  <label for="msg">Mensagem:</label>
  <textarea id="msg" name="msg" rows="5"></textarea>

  <label for="nivel">Nível:</label>
  <select id="nivel" name="nivel">
    <option value="iniciante">Iniciante</option>
    <option value="medio">Médio</option>
    <option value="avancado">Avançado</option>
  </select>

  <button type="submit">Enviar</button>
</form>

Tipos de Input mais comuns

type=""Para quê serve
textCampo de texto simples
emailE-mail (valida formato automaticamente)
passwordPalavra-passe (texto oculto)
numberNúmero
dateData (abre um seletor de datas)
checkboxCaixa de seleção (marcar/desmarcar)
radioOpção única dentro de um grupo
filePermite enviar um ficheiro
submitBotão de envio do formulário

4. O que é CSS?

A sigla CSS significa Cascading Style Sheets — Folhas de Estilo em Cascata. É a linguagem responsável pela aparência visual das páginas web.

Inline

Dentro da própria tag HTML, com o atributo style.

✗ Difícil de manter

Interno

Dentro de uma tag <style> no <head>.

✗ Só funciona nessa página

Externo ✓

Num ficheiro .css separado, ligado com <link>.

✓ Recomendado

<!-- CSS INLINE (não recomendado) -->
<p style="color: red; font-size: 18px;">Texto vermelho</p>

<!-- CSS INTERNO -->
<head>
  <style>
    p { color: red; font-size: 18px; }
  </style>
</head>

<!-- CSS EXTERNO (recomendado) -->
<head>
  <link rel="stylesheet" href="estilos.css">
</head>

5. Seletores CSS — Como Escolher o que Estilizar

Os seletores dizem ao CSS a que elementos aplicar os estilos. A sintaxe é sempre: seletor { propriedade: valor; }

/* Seletor de ELEMENTO */
p { color: #555; line-height: 1.7; }
h1 { font-size: 2em; color: #1a1a1a; }

/* Seletor de CLASSE */
.destaque { background-color: yellow; font-weight: bold; }
.card { border-radius: 8px; padding: 20px; }

/* Seletor de ID */
#titulo-principal { color: red; font-size: 3em; }

/* Combinação de seletores */
.card h2 { color: red; }
p, h1, h2 { margin-bottom: 10px; }
💡 Classes vs IDs: Uma class pode ser usada em vários elementos. Um id deve ser único por página. Prefere sempre classes!

6. O Box Model — O Modelo da Caixa

Em CSS, todos os elementos são retângulos (caixas). O Box Model descreve as 4 camadas que compõem cada elemento:

Margin (margem exterior — espaço fora do elemento)
Border (borda/moldura do elemento)
Padding (espaço interior — entre borda e conteúdo)
Content (o conteúdo em si — texto, imagem, etc.)
.minha-caixa {
    width: 300px;
    height: 150px;
    padding: 20px;
    padding: 10px 20px;     /* vertical | horizontal */
    border: 2px solid #333;
    border-radius: 8px;
    margin: 30px auto;      /* vertical: 30px | horizontal: centrado */
}

7. Cores e Tipografia no CSS

Formas de definir cores

color: red;
color: #ff4d4d;
color: rgb(255, 77, 77);
background-color: rgba(255, 77, 77, 0.5);

Propriedades de tipografia

p {
    font-family: 'Segoe UI', Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    font-style: italic;
    line-height: 1.6;
    text-align: center;
    text-decoration: none;
    color: #333;
    letter-spacing: 1px;
}
